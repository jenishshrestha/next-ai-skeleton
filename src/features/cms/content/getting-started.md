# Welcome to BuildNext

BuildNext is a professional Next.js starter designed for architectural purity and developer velocity.

## Key Features

- **Feature-Driven Design (FDD)**: Stop the spaghetti code. Organize your features into self-contained modules.
- **Drizzle ORM**: Type-safe database interactions with Neon/PostgreSQL.
- **Better Auth**: Modern, standardized authentication.
- **SVGR & SvgIcon**: Clean icon management.
- **Production-Ready SEO**: Centralized metadata and JSON-LD.

## How to use this starter

1.  **Clone the repo**
2.  **Install dependencies**: `bun install`
3.  **Run dev server**: `bun dev`

Enjoy building!
